Pagination

Pagination
Pagination component enables list components such as Page List, Event List and File list to be paginable. This component allows to navigate through list of pages. It shows number of pages in the list, each of them allowing to go to the associated page and additional buttons which allows to switch to the beginning, the end and previous and next elements of the list. 

Example 1: Default pagination configuration

  • Apple
  • Advocado
  • Banana
If you don't specify the page collapsing threshold you will get the simplest navigation with no page numbers. The most basic thing.
First Previous Next Last

Example 2: Collapsed pagination

My pagination does not show all of the page numbers. This can be achieved by setting collapse pagination threshold (here set to 3). This means that at maximum 3 page numbers will show that are in the immediate vicinity to the selected page.
  • Apple
  • Advocado
  • Banana
  • Cherry
  • Durian
First Previous 1 2 3 ... Next Last

Example 3: Two pagination components per list

First Previous 1 2 3 4 5 6 7 8 Next Last
  • Ribes
  • Strawberry
  • Watermelon
First Previous 1 2 3 4 5 6 7 8 Next Last

With the threshold at 10 (since we have less than 10 pages) you see all the page numbers. Additionally you see that each page list can have multiple Pagination components  attached to it. 

Features and Options
  • Multiple pagination components on page
  • Multiple paginations components per list
  • Multiple configuration options like collapse pagination threshold
