Page List

Page List

The Page List component that renders... a list of pages. Simple. The listing content is generated from fields on pages being listed. You can configure which fields should be shown, and style them any way you want. The pages list is constructed using a Sitecore query, so you can choose if you would like to display child or sibling pages or items with the same template... or any other query you can construct. 

Example 1: Simple list of pages

  • Field Editor
    There are pages, there are products and there are pages about products. Products have specifications that need to be listed. This can be done best with Field Editor.
  • Page Content
    Your content is stored as fields on Sitecore pages. The title, main copy, or header image are just a few examples of fields you might have. With Page Content component you can drag & drop the content wherever you think it fits best and thanks to rendering variants you can easily select the appropriate field to be shown from a drop down list.
  • Page List

    The Page List component that renders... a list of pages. Simple. The listing content is generated from fields on pages being listed. You can configure which fields should be shown, and style them any way you want. The pages list is constructed using a Sitecore query, so you can choose if you would like to display child or sibling pages or items with the same template... or any other query you can construct. 

  • Pagination
    Pagination component enables list components such as Page List, Event List and File list to be paginable. This component allows to navigate through list of pages. It shows number of pages in the list, each of them allowing to go to the associated page and additional buttons which allows to switch to the beginning, the end and previous and next elements of the list. 
  • Plain HTML

    A generic component that allows you to include a piece of HTML markup directly. This comes in handy when you need to do something very specific, like embedding a third-party component (widget, tracking script, etc.), or when none of the available components suits your custom needs.

In this example we are displaying a list of child pages from a given page in our site. We are using one of the out of the box variants which is displaying page title and page description fields.

Configuration:

  • Start page is set to the parent of this page
  • Set to show the children of that page
  • Selected component variant is showing page title and content
  • List of items is limited to show just 5 elements
