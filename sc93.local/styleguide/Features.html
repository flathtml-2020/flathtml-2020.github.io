Features

Learn about all the features that SXA offers on top of the Sitecore Experience Platform 
Features

Sitecore Experience Accelerator (SXA) accelerates deployment of responsive websites quickly and cost effectively, minimizing reliance on technical skills while improving maintenance of governance standards.

This tool uniquely separates front-end and back-end development enabling the parallelisation of design, content, and development. SXA has an extensive library of re-usable themes, templates and over 50 flexible components allowing you to achieve any layout imaginable.

As well, SXA features the innovative Creative Exchange which accelerates and simplifies sharing of website design files between development roles and agencies. SXA provides 90% of the functionality required to deploy large numbers of websites out of the box.

  • Components

    Functionality built once: skinnable, configurable, and reusable

  • Context Switching

    Demo switching is a really helpful feature, which provides for author possibility to set up data source once for parent container or component.

  • Creative Exchange

    Empowering your Creative & Front-End developers

  • Drag And Drop Editing

    What easier way to construct the content for the page you are working on than to simply drag components off the shelf and put them directly where you need them? If you need to structure the layout of your page, you can also drag and drop column and row splitters to divide the content up correctly. No need to go to a developer.

  • Layout

    Layouts in SXA are built using grid systems like Bootstrap, Foundation or our Responsive Web Design version of 960 Grid System. The platform ships with just one single Sitecore layout but allows you to build your custom designs from it.

  • Multimarketing

    Thanks to the multi-tenant nature of SXA architecture, clients are able to run multiple micro-sites along their main sites on a single instance of Sitecore

  • Page Designs

    Manage your various layouts and page furniture on platform

  • Partial Designs

    Manage and reuse your design elements on platform

  • Rendering Variants

    Like it says on the tin: Rendering Variants are variations of components. When a user places a component on a page, they are able to select different ways that the component may appear or what content it shows. This might vary quite a lot from project to project, so we have created a facility for a "super user" to create their own Rendering Variants. This avoids the need for building specific components which are only slight variations of existing components.

  • Site Delivery Workflow

    Sitecore Experience Accelerator is not just an extension of the Sitecore interface, but a new way of approaching the production of websites.

  • Sticky Notes

    Sitecore Experience Accelerator is all about collaboration

  • xDB Integration

    Get personal with your visitors
